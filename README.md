# Skippedx0r - Cybersecurity Portfolio

Repositori ini memuat *source code* untuk halaman portofolio pribadi saya. Situs ini dirancang untuk menunjukkan rekam jejak, keahlian, dan *write-ups* terkait **Cyber Security** dan **Competitive Programming**.

## 🛠 Tech Stack
- **HTML5 & CSS3** (Custom Hacker/Terminal aesthetics dengan efek Scanline)
- **Vanilla JavaScript** (Interaktivitas DOM & efek Typewriter Terminal)

## 📌 Fitur
- **Terminal Simulator:** Animasi hero section yang mengetik otomatis seperti CLI Kali Linux.
- **Project Showcase:** Portofolio *Reverse Engineering*, eksploitasi Kriptografi, desain Database, serta repositori CP.
- **Responsif:** Optimal di perangkat *mobile* maupun *desktop*.

> *"Security is a process, not a product."*
